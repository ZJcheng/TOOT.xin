Tomcat

Tomcat 搭建

本机环境：centos

安装 gcc 以及 lrzsz

yum -y install gcc* 安装编译源码包所需的软件
yum -y install lrzsz 安装时候可以通过 xshell 终端进行上传下载的一些操作

安装 JDK:解压 JDK 软件包 tar -zxvf jdk…..tar.gz

mv jdk1.7.0_25 /jdk

配置 JDK  的环境变量

vim /etc/profile

刷新配置文件：source /etc/profile

安装 Tomcat

解压 Tomcat 软件包并拷贝到 /usr/local 目录下

mv apache-tomcat-7.0.42 /usr/local/Tomcat

配置 Tomcat 的环境变量

source /etc/profile 刷新环境变量使其生效

启动 tomcat 程序

chmod 777 /usr/local/Tomcat/bin/*.sh

tomcat 开启： /usr/local/Tomcat/bin/catalina.sh start

Tomcat端口默认为8080

修改稿端口：vim /usr/local/Tomcat/conf/server.xml

Tomcat 多实例

拷贝原来的Tomcat到另外一个目录，如：tomcat-8090，清除logs目录

rm -rf /usr/local/Tomcat/logs/

cp -a /usr/local/Tomcat /usr/local/Tomcat8090

修改Tomcat中的/conf/server.xml文件，把shutdown和Connector端口修改成另外的数值，如8006/8090

vim /usr/local/Tomcat/conf/server.xml

修改startup.sh和shutdown.sh文件，在第一行加入：export CATALINA_HOME=/tomcat路径

vim shutdown.sh

vim startup.sh

另一个实例

vim /usr/local/Tomcat8090/conf/server.xml

vim /usr/local/Tomcat8090/bin/startup.sh

vim /usr/local/Tomcat8090/bin/shutdown.sh

如果多个实例就添加多个实例。

启动tomcat

/usr/local/Tomcat/bin/catalina.sh start

/usr/local/Tomcat8090/bin/catalina.sh start

部署 war  包

部署 Java 的 WAR 包

在 /usr/local/tomcat/conf/server.xdl 配置文件中的 server 区域中添加

vim /usr/local/Tomcat/conf/server.xml

将 war包 拷入到 /usr/local/tomcat/webapps 目录下，并删除 ROOT 目录

重启服务
